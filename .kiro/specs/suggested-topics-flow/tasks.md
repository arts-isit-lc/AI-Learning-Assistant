# Implementation Plan: Suggested Topics Flow

## Overview

Implement the improved topic suggestion workflow across the New Module and Edit Module pages. Create shared utility functions for auto-generation gating and topic merging, wire auto-generation on the New Module page with direct population, and replace the overwrite-style "Populate" button on the Edit Module page with an additive "Suggested Topics" merge panel.

## Tasks

- [x] 1. Create shared utility functions
  - [x] 1.1 Create `frontend/src/utils/topicGenerationHelpers.js` with `shouldAutoGenerate`, `mergeTopics`, and `findDuplicates`
    - Implement `shouldAutoGenerate(trackedFiles)` — returns true when all entries are terminal AND at least one is `complete`
    - Implement `mergeTopics(existing, incoming)` — case-insensitive dedup, returns `[...existing, ...newOnes]`
    - Implement `findDuplicates(existing, incoming)` — returns Set of incoming topics whose lowercased-trimmed form matches existing
    - Export all three functions
    - _Requirements: 1.1, 2.5, 4.3, 5.1_
